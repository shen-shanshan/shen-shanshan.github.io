# Shanshan's blog

Welcome to my personal [<u>blog</u>](https://shen-shanshan.github.io/)! :smile:

## Commands

publish a new post:

```cmd
hexo new -p <dir_name>/<post_name> "post title"
```

> [!NOTE]
> post_name 不带 `.md` 后缀，post_name 中间有空格时使用双引号包裹，如：dir_name/"post name"。
