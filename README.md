# Shanshan's Blog

Welcome to my personal [<u>blog</u>](https://shen-shanshan.github.io/)! :smile:

## Quick Start

Publish a new post:

```bash
hugo new posts/title.md
```

Preview at localhost:

```bash
hugo server
```

Build site:

```bash
hugo -t blowfish
# or: hugo build
```

> [!NOTE]
>
> Remember to add `{{< katex >}}` just once and convert symbols for math syntax in markdown files:
>
> - `\\` -> `\\\`
> - `_` -> `\_`
